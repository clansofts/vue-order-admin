# vue-order-admin

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### 发布测试版本
yarn build-test

### 运行
yarn serve

### 代码规范检测及修复
yarn lint

### 发布正式环境
yarn build



#提交代码之前，先执行一下 yarn lint 和 yarn build-test
